## :milky_way: Introduction

**Victorware** is a **lag-switch** Roblox script designed to make the game lag, which can be of great help for tower defense games like Anime Adventure, All Star Tower Defense, etc.

Press `RightShift` to open or close Victorware. For the best experience, a strong internet is recommended!

### :pushpin: Latest release

Runs the latest release build of Victorware.

```lua
loadstring(game:HttpGetAsync("https://raw.githubusercontent.com/bach2206/Victorware-Lag-Switch/main/Victorware.lua"))()
```
